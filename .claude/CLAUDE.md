# GraphQL Practice Project

GraphQLの使い方を学ぶためのプロジェクトです。

## 目的
- GraphQLの基本を理解する
- シンプルなTodoアプリを通じて学ぶ

## 技術スタック
- **言語**: TypeScript
- **GraphQL Server**: Apollo Server
- **Web Framework**: Express
- **Database**: SQLite
- **ORM**: Prisma（予定）

## アーキテクチャ
```
Client → Apollo Server (GraphQL) → SQLite (DB)
```

## 学習予定
1. GraphQLの基本概念（Query, Mutation, Schema）
2. Apollo Serverのセットアップ
3. SQLiteとの連携
4. Todoアプリの実装（CRUD操作）
